## Сеттинг

Футбольные хулиганы дерутся на поляне в городском парке. Жанр — «забив» (организованная групповая драка по правилам фанатской субкультуры).

**Поле боя:** Поляна в городском парке, трава, деревья по краям.

**Фоновые персонажи (не играбельные, создают атмосферу):**
- Дед
- Мама с коляской
- Велосипедист
- 2 человека с разных сторон с камерой (обязательно!)
- «Тренеры» и «смотрящие» по краям

**Визуальный стиль:** Реалистичный, референсы — реальные видео забивов. Парни в одинаковых футболках, спортивных штанах, разные по характеру и характеристикам.

## Игровая механика

### Уровни системы (Dota Underlords-style)

**Уровень 1 — Фракция (выбирается до матча):**
Игрок выбирает «фирму» — фанатскую группировку клуба. Даёт пассивные бонусы всей команде.  
⚠️ *Фракции и их баланс — TBD. Вернуться к этому отдельно.*  
Варианты: вымышленные названия клубов (юридически безопаснее, чем реальные).

**Уровень 2 — Архетип (тип юнита):**
- **Подписной спортик** — технарь, боец с базой единоборств
- **Отморозок** — непредсказуемый, высокий риск/награда
- **Выездной** — опытный, видел многое

**Уровень 3 — Класс (конкретный юнит внутри архетипа):**

| Архетип | Классы |
|---|---|
| Подписной спортик | МСМК по боксу, Чемпион Беллатура, Двоюродный брат из спецназа, Знакомый боксерчик |
| Отморозок | Сидевший, Бешеный торч, Детдомовец, БС |
| Выездной | Боевой ультрас, Пьянь |

*(Архетипы и классы неполные — пользователь продолжит после обсуждения механики)*

## Подход к разработке

**Выбранный подход:** Форк опенсорс Unity autochess-проекта + замена визуала на дизайн ПОЛЯНЫ.

**Опенсорс основа:** Не выбрана — нужно найти при следующем запуске.

**Разработка через агентов:** Spec разделён по зонам ответственности — каждый блок может взять отдельный агент. Агенты координируются через Claude Code.
